Overview

Angular Smilies module exposes a filter and a directive for smilies parsing (replaced by an image) as well as a directive to display a "smiley picker".

Installation


    
Dependencies

    
ngSanitize

    

        In order to use the filters (which outputs HTML) you will need the ngBindHtml directive which is
        part of ngSanitize module.
    


    
AngularUI Bootstrap or Angular Strap

    

        The smiley picker uses Bootstrap Popover as Angular module. You can use either
        AngularUI Bootstrap or
        Angular Strap.
    


    
Includes

    

        The module is available on Bower. Add angular-smilies to your bower.json file or
        install it manually with bower install angular-smilies --save.

Usage


    
Bootstrap

    
Add the necessary modules to your app definition.

    
{% highlight javascript %}
var app = angular.module('app', [
  'ngSanitize',
  'ui.bootstrap', // OR mgcrea.ngStrap
  'angular-smilies'
]);
{% endhighlight %}

Smiley picker

    

        The module includes a smiley selector based on Bootstrap Popover. It allows to display available smilies and
        append the corresponding code to a model.
