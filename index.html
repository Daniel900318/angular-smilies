Overview

Installation


  
Dependencies

  
ngSanitize

  
In order to use the filters (which outputs HTML) you will need the ngBindHtml directive which is part of ngSanitize module.


  
Angular Strap & AngularUI

  
The smiley picker uses Bootstrap Popover as Angular module. You can use either AngularUI or Angular Strap.


  
Includes

  
The module is available on Bower. Add angular-smilies to your bower.json file or install it manually with bower install angular-smilies.

Usage


  
Bootstrap

  
Add the necessary modules to your app definition.

{% highlight javascript %}
var app = angular.module('app', [
  'ngSanitize',
  'ui.bootstrap', // OR mgcrea.ngStrap
  'angular-smilies'
]);
{% endhighlight %}
