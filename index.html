Overview

Angular Smilies module exposes a filter and a directive for smilies parsing (replaced by an image) as well as a directive to display a "smiley picker".

Installation


  
Dependencies

  
In order to use the filters (which outputs HTML) you will need the ngBindHtml directive which is part of ngSanitize module.

  
The smiley picker use AngularUI Popover module, BUT the official release does not support yet templates as Popover content, thus you will need the fork of J Bruni. Angular Smilies will be updated as soon as the feature is integrated in the official release.


  
Includes

  
The module is available on Bower. Add angular-smilies to your bower.json file or install ir manually with bower install angular-smilies.

Usage


  
Bootstrap

  
Add the necessary modules to your app definition.

{% highlight javascript %}
var app = angular.module('app', [
  'ngSanitize',
  'ui.bootstrap',
  'angular-smilies'
]);
{% endhighlight %}
